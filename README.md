# 📖 About the Project

**CodeAlpha AI Cloud Chat Bot** is a modern AI-powered conversational web application developed using **FastAPI**, **Google Gemini AI**, **HTML**, **CSS**, and **JavaScript**.

The application enables users to interact with Google's powerful Gemini AI model in real time through an elegant and responsive web interface. It is designed to provide fast, intelligent, and natural conversations while maintaining a clean user experience.

This project demonstrates the integration of modern AI technologies with web development and showcases practical implementation of REST APIs, backend development, frontend design, and AI model integration.

---

# 🎯 Project Objectives

The main objective of this project is to build a complete AI chatbot capable of:

- Answering user questions intelligently
- Providing real-time AI responses
- Maintaining conversation flow
- Delivering a responsive and user-friendly interface
- Demonstrating integration of Google Gemini API with FastAPI

# ⚙ Installation Guide

## Step 1

Clone the repository

```bash
git clone https://github.com/mohammadzunaid333-hash/CodeAlpha-Making-a-Chat-Bot.git
```

---

## Step 2

Move into the project

```bash
cd CodeAlpha-Making-a-Chat-Bot
```

---

## Step 3

Create a Virtual Environment

```bash
python -m venv venv
```

---

## Step 4

Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## Step 5

Install Dependencies

```bash
pip install -r backend/requirements.txt
```

---

## Step 6

Create a **.env** file

```env
GEMINI_API_KEY=YOUR_GOOGLE_GEMINI_API_KEY
```

---

## Step 7

Run the Backend

```bash
uvicorn backend.app:app --reload
```

---

## Step 8

Open the Frontend

Simply open

```
frontend/index.html
```

or run it using **Live Server** in VS Code.
