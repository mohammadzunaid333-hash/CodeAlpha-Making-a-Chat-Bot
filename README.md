# 🚀 CodeAlpha-Making a Chat Bot

An intelligent AI-powered chatbot built using **FastAPI**, **Google Gemini AI**, **HTML**, **CSS**, and **JavaScript**. The chatbot provides real-time AI responses with a modern UI and smooth user experience.

---

## ✨ Features

- 🤖 Google Gemini AI Integration
- ⚡ FastAPI Backend
- 💬 Real-time Chat
- 📝 Chat History
- 📋 Copy Response Button
- 🌙 Dark Theme UI
- 📱 Responsive Design
- ⚡ Typing Animation
- 🎨 Modern Interface

---

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/mohammadzunaid333-hash/CodeAlpha-Making-a-Chat-Bot.git
```

Move into project

```bash
cd CodeAlpha-Making-a-Chat-Bot
```

Install dependencies

```bash
pip install -r backend/requirements.txt
```

Create a `.env` file inside the backend folder.

```
GEMINI_API_KEY=YOUR_API_KEY
```

Run the backend

```bash
cd backend
python -m uvicorn app:app --reload
```

Open

```
frontend/index.html
```

using Live Server.
